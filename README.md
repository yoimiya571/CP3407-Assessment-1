# Reading App

## Project Overview

Reading App is a web-based reading community platform designed to help users track their reading journey, share reading experiences, and interact with other readers through book-based discussion clubs.

The platform allows users to:
- Set personal reading goals
- Save bookmarks and highlights
- Join book discussion clubs
- Share comments and reading experiences
- Interact with other readers in a community environment

This project is developed for CP3407 Software Engineering Project at James Cook University.

## Project Objectives

The objectives of this project are to:

1. Create an online platform for reading communities
2. Encourage users to maintain reading habits
3. Improve interaction between readers through discussion clubs
4. Provide a user-friendly interface for tracking reading progress
5. Deliver a scalable web application using modern development tools
